# Source: https://www.electronjs.org/docs/latest/api/structures/keyboard-input-event

* `type` string - The type of the event, can be `rawKeyDown`, `keyDown`, `keyUp` or `char`.
* `keyCode` string - The character that will be sent
as the keyboard event. Should only use valid [Accelerator](/docs/latest/tutorial/keyboard-shortcuts#accelerators)
key codes.
